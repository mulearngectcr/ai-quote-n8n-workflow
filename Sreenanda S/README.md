AI Quote Coach

A simple n8n workflow that fetches a random motivational quote from the ZenQuotes API, uses AI to explain it in simple language, and sends the result via email.

Workflow

Manual Trigger → HTTP Request → AI Agent (Groq) → Gmail

Features

* Random quote generation
* AI-powered explanation
* Student-friendly example
* Actionable takeaway
* Motivational one-liner
* HTML email formatting
* Timestamp and author included

Tech Stack

* n8n
* ZenQuotes API
* Groq LLM
* Gmail
